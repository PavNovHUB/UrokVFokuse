
# Урок в Фокусе

**Урок в Фокусе** — концепт веб‑платформы для оценки качества уроков и педагогических практик.

## Возможности
- Современный адаптивный интерфейс
- Понятная структура критериев оценки
- Готово для расширения под API и аналитику

## Запуск
Просто откройте `index.html` в браузере  
или используйте GitHub Pages.
